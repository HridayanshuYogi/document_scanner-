# Smart Document Scanner

## Author
Ritik Sharma (2301010304)

## Objective
Analyze sampling and quantization effects on document images.

## Features
- Image acquisition
- Grayscale conversion
- Sampling analysis
- Quantization analysis
- Visual comparison
- Output saving

## Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib

## How to Run
1. Install required libraries:
   pip install opencv-python numpy matplotlib

2. Run the program:
   python scanner.py

## Output
The program processes document images and shows comparison results.
